  • 문진 질문은 의사 선생님이 직접 말하는 스타일(존댓말 · 교정본 원문)로 제시되고, 답변은 환아가 의료진에게 대답하는 느낌으로 설계한다(벼리 역할 → S5 결과 화면 기능정의). 화면 구성(위→아래): ① 프로그레스 바 → ② 벼리 + 말풍선 → ③ 증상명 + 증상 설명 → ④ 답변 카드
  • ① 프로그레스 바 — 문진 시작 시 상단 고정. 전체 진행률(선택 증상 수 × 질문 수 기준)을 답변마다 채움 + "증상 2개 중 1번째 · 질문 1/2" 텍스트 병기
  • ② 벼리 = 질문 안내 — 벼리는 선생님 질문을 안내하는 도우미. 공감 한마디(반말, 판단·평가 없음, 여러 종 로테이션 P-12) 후 선생님 질문(존댓말 원문)을 안내. "딸꾹질 힘들었겠다. 선생님이 아래와 같이 몇 가지 물어보셨어, 같이 답해보자."
  • ③ 증상명 + 설명 병기 — 질문 영역에 증상명 + 「증상 설명 (도움말) 교정본」을 항상 함께 노출 — 아이가 무엇을 묻는지 이해한 상태로 답하게 함. 증상 설명은 벼리가 문진 진행을 이끌며 화면 이미지처럼 알려줌
  • ④ 답변 카드 — 선택형 카드(슬라이더 없음). 탭 즉시 다음 질문 전환(카드 슬라이드·페이드 마이크로 애니메이션으로 정적이지 않게), 환아가 의료진에게 대답하는 느낌. 직전 질문으로 돌아가 답을 고칠 수 있음

  • 벼리 영역 역할 — 벼리는 환아가 기록을 쉽게 진행하도록 안내하고, 기록 내용을 이해하기 쉽게 정리해준다("메신저" 아님). 환아의 기록 행위를 칭찬·공감하고, 한 기록을 정리해서 보여준다("오늘 기록을 이렇게 정리해봤어"). 의료적 판단·처방 언급 금지. 코끼리 화법(이름 호명·판단 없는 직설·또래 목소리)
  • 의사 아이콘 — 기록 요약·간단 관리방법 영역(S5)과 문진 질문(S4)은 의사 아이콘 + 흰 배경 + teal 테두리 말풍선으로 표시해 벼리(크림 배경 + 골드 테두리)와 구분 — 환아가 의료진과 주고받는 방향감 부여 [P-신규-03 픽토그램 스타일 확정 필요]
  • 관리방법 출처 표기 — 타이틀 자체를 의료진 귀속으로: 환아 "병원 선생님께서 #증상명엔 이렇게 하라고 하셨어" / 보호자 "병원 선생님이 알려주는 #증상명 관리법" + 출처 뱃지 1줄 "의료진 제공 관리 지침". "선생님"만 쓰지 않고 "병원 선생님"으로(학교 선생님과 혼동 방지). 의료적 판단·처방은 벼리가 언급하지 않는다(거짓 안심 금지)
  • 공통 안전 안내 — "열이 38도 이상이거나 갑자기 많이 아프면 바로 보호자에게 알려줘. 보호자와 함께 병원에 연락해봐." G0~G4 전 결과 화면 하단에 항상 노출. "의료진에게 전달했다·의료진이 보고 있다"류 표현은 쓰지 않는다(삼성서울병원 의견)
  • 표정 + 신체 감각 언어 표기 — 벼리 표정(5종) 옆에 해당 Grade의 감각 언어를 라벨로 표기(😊 거의 안 느껴져요 / 😐 꽤 불편해요 / 😟 많이 힘들어요 — 여러 증상이면 최고 Grade 기준) + 기록 요약 칩에도 증상별 표정·감각어 병기. 감각어는 라벨·칩 전용 — 벼리 문장 안에 그대로 넣지 않는다(반말·존댓말 혼재 방지)
  • G3+ 공감 멘트 규칙 — 상태를 부정하지 않고 반말로 인정("오늘 몸이 많이 힘들었겠다"). 불안 조장·경고 톤 금지, 거짓 안심도 금지
  • G0 증상없음 — S1에서 "아니, 특별히 없어"를 고르면 문진 없이 바로 이 화면(증상 없음 기록 완료). 고맥락 재확인("정말 없을까?")·피드백 메시지·개인화 문구는 두지 않는다. 벼리 멘트만: "잘 기록했어. 오늘은 힘든 증상이 없구나. 컨디션 잘 유지해줘, 언제든 증상이 생기면 알려줘!" CTA: 홈으로 단일. 증상 없음은 서버에서 G0로 저장(환아 화면 표정 적용) — 이후 컨디션 변화 비교용 기록으로 활용. "정상·안심·안전" 단정 표현 금지 — "잘 기록했어" 중심
  • G1·G2 — 기록 요약(표정+감각어) + 간단한 자가관리 지침(짧게) + "더 보기(증상관리 가이드)". 환아도 최소한의 대처법은 결과 화면에서 바로 확인. [짧은 지침을 어떻게 정리·노출할지는 추후 세부 기획 예정]
  • G3·G4 — 환아 화면 경고색 최소 사용(teal 유지). 벼리: "오늘 몸이 많이 힘들었겠다. 이렇게 기록해줘서 정말 고마워. 보호자한테 오늘 기록 같이 보여줘". 기록 요약 + "관리 방법은 기록보기에서 보호자와 함께". 상세 행동요령 직접 노출은 하지 않음. P-신규-01: 전체 문구 메디컬팀 검수 필수

Grade 표기 원칙 — 왜 환아 화면에서 G라벨을 바꾸는가

환아에게 G0~G4 의료 라벨은 이해하기 어렵고 불안감·과소 보고(소아암 환아의 증상 축소 보고 경향 — 아래 의료 근거)를 유발할 수 있다. 따라서 환아 화면에서는 벼리 표정(5종) + 신체 감각 언어로 표현하고(표정·감각어 매핑 → §03 S5), Grade 자체는 서버·의료진 웹·보호자 기록보기에서 기존 그대로 유지한다. 표정 표기는 환아 모드 전반(S5 결과 · S2 최근 기록 배지 · 홈 기록보기 등)에 일관 적용하며, 홈 영역 표기 상세는 홈 기획서와 함께 결정한다.
